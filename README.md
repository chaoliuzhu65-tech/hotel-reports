## 📦 快速安装

### 方法1：一键安装脚本

```bash
git clone https://github.com/chaoliuzhu65-tech/hotel-reports.git
cd hotel-reports
bash install/install.sh
```

### 方法2：手动复制

```bash
# 宣推SKILL
cp -r skills/xuantui/* ~/.openclaw/skills/

# 市场探针SKILL
cp -r skills/market-probe/* ~/.openclaw/skills/
```

### 方法3：通过lark-cli安装

```bash
lark-cli skills install hotel-xuantui
lark-cli skills install market-probe
```
